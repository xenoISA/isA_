# isA

## Overview

A standalone AI-powered application with multi-app capabilities and session management. The frontend is a Next.js app that routes all backend traffic through the APISIX gateway and supports multiple app modules (Dream, Hunt, Omni, Assistant, DigitalHub, DataScientist, Doc).
